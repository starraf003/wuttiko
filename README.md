# Wuttiko

Prototype web/PWA de l'application Wuttiko : comparateur de prix local au Sénégal.

## Tester en local
```bash
npm install
npm run dev
```

## Déployer sur Vercel
1. Aller sur vercel.com
2. Créer un compte ou se connecter
3. Cliquer sur Add New Project
4. Importer ce dossier via GitHub ou en ZIP selon l'option disponible
5. Framework: Vite
6. Build command: npm run build
7. Output directory: dist

## Tester sur téléphone
Une fois le lien Vercel obtenu :
- Ouvrir le lien sur téléphone
- Menu du navigateur
- Ajouter à l'écran d'accueil
